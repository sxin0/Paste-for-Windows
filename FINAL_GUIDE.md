## 🎮 使用步骤

### 托盘功能演示使用（最推荐）

1. **启动应用**
   ```bash
   python tray_demo.py
   ```

2. **添加测试数据**
   - 点击"添加测试数据"按钮
   - 系统会添加10个不同类型的示例内容

3. **最小化到托盘**
   - 点击窗口的最小化按钮
   - 窗口会隐藏到系统托盘

4. **点击托盘图标**
   - 单击或双击系统托盘图标
   - 底部交互窗口会从屏幕下方冒出来

5. **体验完整功能**
   - 在底部面板中搜索内容
   - 点击项目选择内容
   - 按ESC键关闭面板

### 简化版应用使用

1. **启动应用**
   ```bash
   python simple_app.py
   ```

2. **添加测试数据**
   - 点击"添加测试数据"按钮
   - 系统会添加10个不同类型的示例内容

3. **显示底部面板**
   - 点击"显示底部面板"按钮
   - 观看从屏幕下方冒出来的动画效果

4. **体验搜索功能**
   - 在搜索框中输入关键词
   - 观察实时过滤效果

5. **选择内容**
   - 点击列表中的项目
   - 查看状态栏的反馈

### 完整应用使用

1. **以管理员身份运行**
   ```bash
   python run.py
   ```

2. **自动监听剪贴板**
   - 复制任何内容都会自动保存
   - 双击系统托盘图标显示历史

3. **使用快捷键**
   - 按 `Win+V` 显示底部面板
   - 按 `ESC` 关闭面板

## 🔧 故障排除

### 常见问题

1. **剪贴板访问错误**
   - **解决方案**: 使用简化版应用 `python simple_app.py`
   - **原因**: Windows 剪贴板访问权限问题

2. **导入错误**
   ```bash
   pip install -r requirements.txt
   ```

3. **PyQt6 问题**
   ```bash
   pip install PyQt6>=6.4.0
   ```

### 推荐解决方案

**使用简化版应用** - 这是最稳定的运行方式：
```bash
python simple_app.py
```
